# API Project

This project demonstrates how to interact with a REST API using Python.

## Description

The scripts retrieve employee TODO list information from an API and export the data into different formats such as CSV and JSON.

## Files

0-gather_data_from_an_API.py  
Fetches employee TODO progress from the API.

1-export_to_CSV.py  
Exports employee TODO list to a CSV file.

2-export_to_JSON.py  
Exports employee TODO list to a JSON file.

3-dictionary_of_list_of_dictionaries.py  
Exports TODO lists of all employees to a single JSON file.

## Requirements

- Python 3
- requests module
